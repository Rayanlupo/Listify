
# Make It List

Make It List a lightweight CLI that helps you turning a string into a list directly from the terminal 


## Installation

Install my-project with npm

```bash
 npm i make-it-list
```
    
## Usage

From your terminal simply run: 

```bash
  makeitlist
```

and you should get something like this
